# 组件

## QLabel
* 自身属性
  * 位置、大小、绑定子组件等
* 绑定事件
  * 有参函数（右键、左键、单击、双击）
  * 无参函数
* 绑定信号
  * 有参函数
  * 无参函数

## QTableWeiget
* 自身属性
  * 位置、大小、合并单元格、绑定子组件等，cell_value_changed时 currentRow和column为0
* 绑定事件
  * 有参函数
  * 无参函数
  * 不同row 或col绑定不同的事件
* 绑定信号
  * 有参函数（右键根据col不同事件或菜单）
  * 无参函数
