# 抽象工厂模式
## 定义
定义一个工厂类接口，让工厂子类来创建一系列相互依赖的对象
## 例子
* 生产一部手机，需要手机壳、CPU、操作系统这三类零件。不同品牌手机需要不同的手机壳、CPU以及操作系统。
* 与工厂模式不同，该为每个手机品牌创建工厂类来生产对应的零件并组装成为手机对象。
```mermaid

    <<interface>>
    +show_cpu(self)

class Shell{
    <<interface>>

}
class PhoneFactory{
    <<interface>>
    +make_shell(self)PhoneShell
    +make_cpu(self)CPU
    +make_os(self)OS

class SmallShell{
    +show_shell(self)

class BigShell{

class AppleShell{
    +show_shell(self)

class SnapDragonCPU{
    +show_cpu(self)
}

    +show_cpu(self)
}

    +show_cpu(self)

class Android{
    +show_os(self)

    +show_os(self)

class MIfactory{
    +make_cpu(self)SnapDragonCPU

}
class Applefactory{
    +make_cpu(self)A14CPU
    +make_shell(self)AppleShell

class Huaweifactory{
    +make_cpu(self)MediaTekCPU
    +make_shell(self)BigShell
    +make_os(self)Android
}
```
